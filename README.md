Film Arşivi Web Sitesi
Bu proje, modern web tasarım tekniklerini pekiştirmek amacıyla geliştirilmiştir. Herhangi bir hazır kütüphane (Bootstrap vb.) kullanılmadan, tamamen saf (vanilla) CSS ile yazılmıştır.

Kullanılan Teknolojiler:

1-HTML5 (Semantik etiketler)
2-CSS3 (Flexbox, CSS Grid, Hover Efektleri, Linear Gradients)

Özellikler:

Responsive Tasarım (Mobil Uyumlu)
Interstellar temalı Hero (Vitrin) alanı
CSS Grid ile otomatik hizalanan film kartları
Karanlık Mod (Dark Theme) tasarımı
